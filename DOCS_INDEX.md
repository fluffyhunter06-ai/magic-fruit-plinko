# 📚 Documentation Index

Welcome to Magic Fruit Plinko! Here's a guide to all the documentation files.

## 🎯 Choose Your Path

### 👶 **I Just Want to Play**
1. Run `start.bat` (Windows) or `./start.sh` (Mac/Linux)
2. Game opens in your browser
3. Click LEFT/RIGHT to drop coins
4. **That's it!** Enjoy the game!

### 🚀 **I Want to Deploy to GitHub**
Follow this path:
1. Start with: [`GITHUB_SETUP.md`](#github_setupmd)
2. Then read: [`README.md`](#readmemd) for context
3. Use: [`QUICK_REFERENCE.md`](#quick_referencemd) for commands

### 💻 **I Want to Understand Everything**
Read in this order:
1. [`README.md`](#readmemd) - Game overview
2. [`SETUP_COMPLETE.md`](#setup_completemd) - Setup details
3. [`GITHUB_SETUP.md`](#github_setupmd) - Deployment guide
4. [`QUICK_REFERENCE.md`](#quick_referencemd) - Commands cheat sheet
5. [`FINAL_SUMMARY.md`](#final_summarymd) - Complete summary

### 🔧 **I Want to Modify the Code**
1. Read [`README.md`](#readmemd) for architecture
2. Check [`QUICK_REFERENCE.md`](#quick_referencemd) for build commands
3. Edit files in `src/` folder
4. Run `npm run dev` to test changes
5. Commit with `git add . && git commit -m "message"`

### **start.bat** (Windows)
**Purpose**: One-click game launcher for Windows
**How to use**: Double-click the file
**What it does**:
- Checks if Node.js is installed
- Installs dependencies if needed
- Starts the development server
- Opens game automatically

---

### **start.sh** (Mac/Linux)
**Purpose**: One-click game launcher for Mac/Linux
**How to use**: Run `./start.sh` in terminal
**What it does**:
- Checks if Node.js is installed
- Installs dependencies if needed
- Starts the development server
- Shows access URL

**Q: I just want to play the game**

A: Follow `GITHUB_SETUP.md` to deploy, then share the link
